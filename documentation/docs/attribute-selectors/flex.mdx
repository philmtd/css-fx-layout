# Flex API

This page describes how to use the attribute based flex API.

## Selectors

The flex API consists of two selectors:

| Selector            | CSS               |
|---------------------|-------------------|
| `data-fx-flex`      | `flex: 1 1 0;`    |
| `data-fx-flex-grow` | `flex: 1 1 100%;` |
