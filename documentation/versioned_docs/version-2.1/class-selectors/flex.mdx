# Flex API

This page describes how to use the class based flex API.

## Selectors

The flex API consists of two classes:

| Selector            | CSS               |
|---------------------|-------------------|
| `fx-flex`      | `flex: 1 1 0;`    |
| `fx-flex-grow` | `flex: 1 1 100%;` |
